## Ghost

### Ghost CMS

#### Requirements
- NPM 
- Ghost CLI

#### Install dependencies

```
npm install ghost-cli@latest -g
npm install
```

#### Start application

```
ghost start
```


### Ghost App

#### Requirements
- NPM
- NODE version 14.17


#### Install dependencies

```
npm install
```

#### Start application

```
npm run start
```
